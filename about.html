# About Me

Yasuyuki Kamitokusari [ freelance software engineer / developer ]

1998年より大手SI企業系列の札幌事業所に勤務。
17年のSIプロジェクト経験を経て2016にフリーランスとして独立。
C/C++による開発経験が最も長く、JavaおよびC派生系プログラミング言語(perl, Rubyなど)による開発プロジェクトの業務委託を中心に活動中です。
長期間にわたるシステム保守業務より、プロジェクトの改善活動やレガシーシステムにおけるアジャイルなプラクティス導入などをメインに行ってきました。
特に、リファクタリング、テスト駆動開発、テスティング、システムテストの自動化など、アジャイル開発のテクニカルなプラクティスの導入に強みがあると自負しております。
オブジェクト指向開発、ドメイン駆動設計を業務で実践することに主眼を置き、変更に強く、かつ本質を見失わない、しなやかな設計を心がけています。
